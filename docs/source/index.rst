Welcome to GCC Tiny documentation!
==================================

**GCC Tiny** is a sample programming language showcasing how to add a front end to GCC.

It is converted from Roger Ferrer Ibáñez blog series: 

* `Tiny Part 1: Specification <https://thinkingeek.com/2016/01/05/tiny-gcc-front-part-1/>`_
* `Tiny Part 2: Initial Boilerplate <https://thinkingeek.com/2016/01/06/tiny-gcc-front-part-2/>`_
* `Tiny Part 3: Lexical Analysis <https://thinkingeek.com/2016/01/08/tiny-gcc-front-part-3/>`_
* `Tiny Part 4: Syntactical Analysis <https://thinkingeek.com/2016/01/10/tiny-gcc-front-part-4/>`_
* `Tiny Part 5: Data Required by Semantical Analysis <https://thinkingeek.com/2016/01/16/tiny-gcc-front-part-5/>`_
* `Tiny Part 6: Semantical Analysis and Generics I <https://thinkingeek.com/2016/01/17/tiny-gcc-front-part-6/>`_
* `Tiny Part 7: Semantical Analysis and Generics II <https://thinkingeek.com/2016/01/19/tiny-gcc-front-part-7/>`_
* `Tiny Part 8: Arrays <https://thinkingeek.com/2016/01/30/tiny-gcc-front-part-8/>`_
* `Tiny Part 9: Boolean <https://thinkingeek.com/2016/01/31/tiny-gcc-front-part-9/>`_
* `Tiny Part 10: Type Names <https://thinkingeek.com/2016/09/04/tiny-gcc-front-part-10/>`_
* `Tiny Part 11: Record Types <https://thinkingeek.com/2016/09/05/tiny-gcc-front-part-11/>`_

.. note::

   This project is under active development.
